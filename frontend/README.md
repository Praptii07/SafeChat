# Safe Chat Moderation System

A web application that allows users to send messages while preventing inappropriate or offensive content from being uploaded.

## Features

* Send and view chat messages.
* Detect inappropriate content.
* Block offensive or harmful messages.
* Allow only safe and appropriate chats.
* Simple and responsive user interface.

## Technologies Used

* React.js
* Node.js
* Express.js
* Axios
* CSS

## How It Works

1. The user enters a message.
2. The system analyzes the message.
3. If the message contains inappropriate content, it is blocked.
4. If the message is safe, it is uploaded and displayed in the chat.

## Note

**Only appropriate and safe chat messages are allowed. Inappropriate content cannot be uploaded.**
